LeagueManager scoreboard widget for GameDay theme.

== Description ==

This plugin is ideal for sports websites running GameDay premium theme and most important,  it’s Open Source!

lm-gd-scoreboard takes scores from the popular LeagueManage wp plugin, and shows them in GameDay’s top widget scoreboard.
Scores merged from all leagues managed by LeagueManger plugin, and categorized by days. Website admins can change dates range to be shown on widget.

Features:

* Extends original GameDay’s scoreboard widget look and feel.
* Grab scores from LeagueManager plugin database.
* Categorize scores by date.
* Customizable dates range from widget manager dialog.

Source code could be found on [GitHub](https://github.com/barshai/wp-plugin-lm-gd-scoreboard).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/lm-gd-scoreboard` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add 'Gameday: LeagueManager Scoreboard Widget' to 'Scoreboard widget are' in appearance->widgets
4. You may select your preferred numerical matches dated range in widget's settings.


== Frequently Asked Questions ==

= I can't activate the plugin =

Make sure both LeagueManager plugin and gameday based theme are installed and active on your wordpress installation.
